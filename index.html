HTML

  • HTML Elements are composed of opening and closing tags with angle brackets <>
  • The head Elements are not visible to the user
  • The body element contains the content that is shown to the user
  • Declerations are not case-sensitive; HTML elements are case-sensitive
  • HTML attributes are always defined in the beginning element, never the closing element. Example: Language
  • When working with visible HTML elements, the content is contained between an opening tag and a closing tag
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

CSS

  • Separation of concerns is based on the idea that each section of code should have its own responsibility
  • A margin property indicates how much space we want around the outside of an element
  • The padding property adds space around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • In JavaScript, to declare a variable, we use the var keyword. After the var keyword, we give the variable a name. This name must be unique because it is how we will reference the information stored in the variable in our code.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • At the most basic level, a function is simply a set of instructions that tells the computer how to perform a certain task.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
